# Financial News

新浪全球实时财经新闻 — Next.js + ISR，每 30 分钟自动更新。

## 功能

- 新浪财经 7×24 实时新闻抓取，智能过滤
- 暗色模式自适应，支持手动切换
- 响应式布局，移动端优先
- 下拉刷新 & 一键刷新
- RSS 2.0 / JSON Feed 订阅

## 技术栈

Next.js 16 (Pages Router) · Tailwind CSS v4 · shadcn/ui · next-themes · Lucide React

## 开发

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
npm start
```

## 部署
